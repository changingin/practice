2. let's make thumbnail - with [div]tag

만드는 법 설명 

  
    
  
  
  1.   먼저 [div]태그를 만들어요.   
    2. 
  
     [ div ]
    
     [ /div ]
  
  
  
  
  2.   [div]의 스타일(style) 속성을 써서 배경 사진 (background-image)를 넣어줘요.    
    3. 
  
  
     [ div style="background-image:(url'thumbnail.jpg') ]
     [ /div ]
     
  
  
  
  
  3.   [div]태그에 class값을 설정해요. (.thumbnail)   
    4. 
  
  
       [ div class="thumbnail" style="background-image:url('thumbnail.jpg')" ]
       [ /div ]
  
  
  
  
 
  4.   [ div ] 태그에 배경화면으로 설정된 이미지의 크기(width,height)를 지정하기 위해 
       [style] 태그를 써서 화면에 나타낸다. 
        
       = 그러면 원본 파일의 좌측, 상면이 화면에 나타난다. 
 
    5. 
 
      [style]
        .thumbnail {
          width:150px;
          height:150px;
        
        }
      [/style]
 
 
 
 
 
  5.   원본 이미지의 좌측, 상면부터 설정한 크기만큼 (150px, 150px) 잘라 오는 것이 아니라 
       원본 이미지의 좌측, 상면부터 설정한 비율만큼 (1:1)  축소, 확대해서 
       설정한 이미지크기(150px, 150px)에  맞추어 가져오게 하려면 
       배경이미지 사이즈를(background-size) cover로 설정한다.  
    6. 
 
      [style]
        .thumbnail {
          width:150px;
          height:150px;
          background-size:cover;
        }
      [/style]
      
      
      
      
 
  6.   원본 이미지의 중앙을 설정 비율만큼 가져오고 싶으면, 
       배경 이미지의 포지션을(background-position)을 center로 설정하자. 
       첫번째는 x축의 center, 두번째는 y축의 center이다.  
    7. 
 
     [style]
        .thumbnail {
          width:150px;
          height:150px;
          background-size:cover;
          background-position: center;
        }
      [/style]
