2. let's make thumbnail - with [div]tag

만드는 법 설명 

  
    
  
  
  1.   먼저 [div]태그를 만들어요.   
    2. 
  
     [ div ]
    
     [ /div ]
  
  
  
  
  2.   [div]의 스타일(style) 속성을 써서 배경 사진 (background-image)를 넣어줘요.    
    3. 
  
  
     [ div style="background-image:(url'thumbnail.jpg') ]
     [ /div ]
     
  
  
  
  
  3.   [div]태그에 class값을 설정해요. (.thumbnail)   
    4. 
  
  
       [ div class="thumbnail" style="background-image:url('thumbnail.jpg')" ]
       [ /div ]
  
  
  
  
 
  4.   [ div ] 태그에 배경화면으로 설정된 이미지의 크기(width,height)를 지정하기 위해 
       [style] 태그를 써서 화면에 나타낸다. 
        
       = 그러면 원본 파일의 좌측, 상면이 화면에 나타난다. 
 
    5. 
 
      [style]
        .thumbnail {
          width:200px;
          height:200px;
        
        }
      [/style]
 
 
 
 
 
  5.   원본 이미지의 좌측, 상면부터 설정한 크기만큼 (200px, 200px) 잘라 오는 것이 아니라 
       원본 이미지의 좌측, 상면부터 설정한 비율만큼 (1:1)  축소, 확대해서 
       설정한 이미지크기(200px, 200px)에  맞추어 가져오게 하려면 
         배경이미지 사이즈를(background-size) cover로 설정한다.  
    6. 
 
      [style]
        .thumbnail {
          width:200px;
          height:200px;
          background-size:cover;
        }
      [/style]
      
      
      
      
 
  6.   원본 이미지의 중앙을 설정 비율만큼 가져오고 싶으면, 
       배경 이미지의 포지션을(background-position)을 center로 설정하자. 
       첫번째는 x축의 center, 두번째는 y축의 center이다.  
    7. 
 
     [style]
        .thumbnail {
          width:200px;
          height:200px;
          background-size:cover;
          background-position: center center;
        }
      [/style]
      
      
      
      
 
  7.   테두리가 둥근 썸네일을 만들어 보자.
      먼저 class 값이 .thumbnail round인 [div]태그를 하나 더 만들자.
 
 
    8. 
   
      [div class="thumbnail round" style="background-image:url('thumbnail.jpg')" ]

    
 
 
 
 
 
  8.  class 값에 .thumbnail 과 .round 값이 있는 [div]태그의  
      테두리 반지금 (border-radius)값을 20px정도 줘 보자. 
    9. 
      
      
      [style]
        .thumbnail.round {
          border-radius:20px;
        }
      [/style]
  
  
  
  
  
  
  9.    테두리가 원인 썸네일을 만들어 보자.  
       먼저 class 값이 .thumbnail round인 [div]태그를 하나 더 만들자.
   
    10. 
  
     [div class="thumbnail circle" style="background-image:url('thumbnail.jpg')" ]

  
  
  
  
  
  
  
  
  10.    class 값에 .thumbnail 과 .circle 값이 있는 [div]태그의  
          테두리 반지금 (border-radius)값을 100%로 설정하자.  
    11. 
  
   [style]
        .thumbnail.circle {
          border-radius:100%;
        }
   [/style]
  
  
  
  
  
  11.    만약, 썸네일들에 테두리(border) 효과를 주고 싶으면   
          .thumbnail 에서 border: 두께, 종류, 색상;   순서로 
  
    12. 
   
   [style]
        .thumbnail {
          width:200px;
          height:200px;
          background-size:cover;
          background-position: center center;
          border: 2px solid gray;
        }
      [/style]
