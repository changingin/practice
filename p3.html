  2.  [img]을 여러 기둥 형태로 만들기 위해 [div]를 사용한다.  
    3. 
  
  [div]tag로 [figure]tag들을 감싸고
  [div]tag의 ID 값을 "columns"로 설정한다.
  
  
  
  [div id="columns"]
 
  [figure]
 [img src="air.jpg"]
 [figcaption] air [/figcaption]
 [/figure]
  .
  .
  .
  [figure]
 [img src="water.jpg"]
 [figcaption] water [/figcaption]
 [/figure]
 
  [/div]
  
  
  
  
  
  3.  각각의 기둥의 넓이를 350px 정도로 설정한다.
    4. 
  
  [div]tag의 기둥 폭을 350px로 설정한다.
  (화면창의 크기가 변해도 한 기둥의 폭이 350px로 고정된다.) 
  
  [style]
   #columns {
   column-width:350px;
   }
  [/style]
  
  
  
  
  
  4.  이미지와 텍스트가 분리되지 않게 설정한다.  
    5. 
  
 [div]tag 안에 있는  [figure]tag의 display를 inline-block으로 설정한다.
  
   [style]
   #columns {
   column-width:350px;
   }
   
    #columns figure{
    display:inline-block;
    }
  [/style]
