3. let's make 'masonry layout' with [figure]tag

만드는 법 설명 

  
    
  
 
  
  1.  화면에 사진들과 사진 설명을 넣는다. 
    2. 
  
* [figure]tag와 [img]tag로 사진을 넣고 
  [figcaption]tag를 사용해 사진설명을 넣는다.

  2.  [img]을 여러 기둥 형태로 만들기 위해 [div]를 사용한다.  
    3. 
  
  [div]tag로 [figure]tag들을 감싸고
  [div]tag의 ID 값을 "columns"로 설정한다.
  
  
  
  [div id="columns"]
 
  [figure]
 [img src="air.jpg"]
 [figcaption] air [/figcaption]
 [/figure]
  .
  .

  [figure]
 [img src="water.jpg"]
 [figcaption] water [/figcaption]
 [/figure]
 
  [/div]
  
  
  
  
  
  3.  각각의 기둥의 넓이를 350px 정도로 설정한다.
    4. 
  
  [div]tag의 기둥 폭을 350px로 설정한다.
  (화면창의 크기가 변해도 한 기둥의 폭이 350px로 고정된다.) 
  
  [style]
  
   #columns {
   column-width:350px;
   }
   
  [/style]
  
  
  
  
  
  4.  이미지와 텍스트가 분리되지 않게 설정한다.  
    5. 
  
 [div]tag 안에 있는  [figure]tag의 display를 inline-block으로 설정한다.
  
   [style]
   
    #columns figure{
    display:inline-block;
    }
    
  [/style]
  
  
  
  
  
  5.   이미지가 기둥에 딱 들어가도록 이미지 크기를 설정한다.  
    6. 
    
    
    
    * 이미지가 깨지는 등 문제가 생겼을 때, 해결하기 위해 먼저 각각의 [tag]들이 
      화면에서 차지하는 부피를 알기 위해 테두리를 치는 습관을 기르자. 
      
  ex)    
    [tag] {
     border:5px solid green;
     }
    
    
    
    
    
    - 기둥 안의 [figure]tag안의 [img]tag의 크기를 100%로 설정해. 
     figure 안으로 img를 딱 넣어준다. 
     
  [style]

     #columns figure img{
        width:100%;
       }
       
  [/style]
  
  
  
  
  
  6.  이미지들 간의 거리를 좁혀주기 위해 [figure]의 margin값을 조절한다. 
    7. 
  - 이미지간 간격이 떨어져 있으면 보기 싫다. 
  - 왜 떨어져 있는지 검사하기 위해 '우클릭' > '검사'를 해서 보면 
   [figure]는 기본적으로 "상하 margin = 16px, 좌우 margin = 40px "을 갖고 있다.
  - margin값을 '0'으로 설정한다.
  
  [style]
   
    #columns figure{
    display:inline-block;
    margin:0;
    }
    
  [/style]
  
    
  
  
  
  7.  세밀하게 column 간의 간격 조절하기 
    8. 
  
  -margin값을 없애주고 나서도 column간의 간격을 조절하고 싶으면 'column-gap'을 사용한다.
   우선 15px을 적용시켜 보자.
   
  [style]
  
   #columns {
   column-width:350px;
   column-gap:15px;
   }
   
  [/style]
  
  
  
  
  
  8.  [frame]사이의 간격도 조절해 보자. 
    9. 
  
  - column 과 column 의 간격은 조절 됐지만 [frame]사이의 간격이 붙어있다.
  - [frame]에 margin-bottom 값을 15px정도 설정해서 간격을 두자.
  
   [style]
   
    #columns figure{
    display:inline-block;
    margin:0;
    margin-bottom:15px;
    }
    
  [/style]
  
  
  
  
  
  
  
  9.  [figure]의 테두리를 꾸며 보자 
    10. 
  
  - 테두리 값으로 1px , solid, rgba(0,0,0,0.2) 으로 설정해서 검은색에 투명도를 주자.
  
  
  
  [style]
  
    #columns figure{
    display:inline-block;
    margin:0;
    margin-bottom:15px;
    border:1px solid rgba(0,0,0,0.2)
    }
    
  [/style]
  
  
  
  
  
  10.  [img] 와 [figcaption] 사이에 줄을 긋자 
    11. 
  
  - [figcaption] 위 테두리에 선을 넣으면 된다.
  
  [style]
  
    #columns figure figcaption{
       border-top:1px solid rgba(0,0,0,0.2)
    }
    
  [/style]
