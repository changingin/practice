1. let's make two botton - with [a]tag

만드는 법 설명 

 


1. 먼저 Blue 라는 링크와, Red 라는 링크를 각각 만든다 

  [a href="#blue"] blue [/a]
  [a href="#red"] red [/a]



2. 링크를 버튼 모양으로 만들어 보자. 
 - 먼저 각 링크에 클래스 값을 준다. 
 
  [a class="btn blue" href="#blue"] blue [/a]
  [a class="btn red"  href="#red"] red [/a]



3. 두 버튼(.btn)이 전형적인 링크처럼 보이지 않기 위해 밑줄을 없애 준다. 

[head] 
 [style]
  .btn {
   text-decoration:none;
  }
 [/style]
[/head] 




4. 두 버튼 (.btn)의 폰트 크기가 너무 작아서, 
  기본 크기인 16px(1rem)의 두 배인 32px(2rem)로 폰트 크기를 변경한다. 
 
 [style]
  .btn {
   text-decoration:none;
   font-size:2rem;
 }
 [/style]
 
 




5.'blue'버튼(.btn.blue) 의 배경 컬러를 blue로 설정한다. 

[style]
  .btn {
   text-decoration:none;
   font-size:2rem;
 }
  .btn.blue{
   background-color:#1f75d9;
  }
 [/style]






6. 두 버튼(.btn) 텍스트 글꼴이 파란색이라서 흰색으로 바꿔준다. 

  .btn {
   text-decoration:none;
   font-size:2rem;
   color:white;
 }
  
  
  
 


7. 두 버튼(.btn)의 배경색 테두리와 글자 사이의 간격을 padding을 활용하여 적용한다. 
  (상 , 우,  하,  좌  ) 순서로 크기px를 적용하자. 

  .btn {
   text-decoration:none;
   font-size:2rem;
   color:white;
   padding: 10px 20px 10px 20px;
 }
 



8. 두 버튼(.btn)이 화면 위쪽에 딱 달라붙는 게 싫어서 
모니터 화면과 버튼의 거리 (margin)을 20px로 설정하자. 

  .btn {
   text-decoration:none;
   font-size:2rem;
   color:white;
   padding: 10px 20px 10px 20px;
   margin:20px;
}
  



9.  버튼(.btn)으로 만들
    [a] tag는 inline level의 tag라서 margin값이 (좌,우)만 적용되고
    (상,하)는 적용되지 않으므로 두 버튼 (.btn)의 display를 inline-block으로 바꿔준다. 
    
    * display를 block으로 바꾸면 blue버튼과 red 버튼이 줄바꿈이 되면서
     좌우로 배열되지 않고 상하로 배열되기 때문이다. 
     

  .btn {
   text-decoration:none;
   font-size:2rem;
   color:white;
   padding: 10px 20px 10px 20px;
   margin:20px;
   display:inline-block;
}





10. 두 버튼(.btn)의 모서리를 둥글게 하기위해서 모서리(border)의 반지름(radius)을 10px정도 적용한다. 
 
  .btn {
   text-decoration:none;
   font-size:2rem;
   color:white;
   padding: 10px 20px 10px 20px;
   margin:20px;
   display:inline-block;
   border-radius:10px;
}


 


11. 두 버튼(.btn)을 클릭했을 때 눌리는 효과를 주기 위해서, 클릭했을 때 버튼의 위치를 바꿔줘야한다. 
 * class값이 btn인 버튼들이 눌렸을 때 (active) 변화를 주는데 (transform) 
   아래로 (Y방향) 10px정도 이동(translate)하게 한다.
   
   .btn:active {
    transform:translateY(10px);
   }
   
   
   
   
   
12. 두 버튼 (.btn) 움직임을 부드럽게 하기 위해서 장면전환(transition)효과를 적용한다. 
   모든 속성에 대해서 (all) 변경되었을 때 0.1초(0.1s) 동안 
   장면전환 (transition)을 적용한다. 


  .btn {
   text-decoration:none;
   font-size:2rem;
   color:white;
   padding: 10px 20px 10px 20px;
   margin:20px;
   display:inline-block;
   border-radius:10px;
   transition: all 0.1s;
}
 
 
 
 
13. blue 버튼(.btn.blue)에 입체감을 주기 위해서 버튼 아래에(bottom) 테두리(border)를 5px두께 만큼 
    보통 실선 (solid)으로설정하고 컬러(color)를 버튼의 색보다 더 진한색(#165195)으로 설정한다. 
 
   .btn.blue{
   background-color:#1f75d9;
   border-bottom:5px solid #165195;
  }
 




14. blue버튼(.btn.blue)을 눌렀을 때 (active) 아래(bottom) 테두리(border)의 두께가 얇아지면 (기존5px > 2px)
    눌려지는 느낌을 줄 수 있다. 

 .btn.blue:active {
  border-bottom:2px solid #165195;
 }





15.더 정교한 효과를 위해 버튼(.btn)을 눌렀을 때 , (active) Y축으로 움직이는(translateY) 거리를
    버튼(.btn) 아래(bottom) 테두리(border)가 얇아지는 정도(5px-2px)와 같게 설정해 보자. 
   
   .btn:active {
    transform:translateY(10px   >   3px );
   }
 




16. 두 버튼의 (.btn) 텍스트가 버튼에 세겨진 느낌을 주기 위해서 텍스트(text)에 그림자 (shadow)를 넣어준다. 
    좌 (-x ) 우 ( x )   /   상 (-y)  하 (y) 거리 값을 설정하고 그림자 색을 검은색(black)으로 설정한다. 


  .btn {
   text-decoration:none;
   font-size:2rem;
   color:white;
   padding: 10px 20px 10px 20px;
   margin:20px;
   display:inline-block;
   border-radius:10px;
   transition: all 0.1s;
   text-shadow: 0px -2px black;
   
}
