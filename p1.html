1. let's make two botton - with [a]tag

만드는 법 설명 

 
1. 먼저 Blue 라는 링크와, Red 라는 링크를 각각 만든다 

  [a href="#blue"] blue [/a]
  [a href="#red"] red [/a]
  
2. 링크를 버튼 모양으로 만들자. 
 - 먼저 각 링크에 클래스 값을 준다. 
 
  [a class="btn blue" href="#blue"] blue [/a]
  [a class="btn red"  href="#red"] red [/a]
  
3. 버튼이 전형적인 링크처럼 보이지 않기 위해 밑줄을 없애 준다. 

[head] 
 [style]
  .btn {
   text-decoration:none;
  }
 [/style]
[/head] 

4.폰트 크기가 너무 작아서, 기본 크기인 16px(1rem)의 두 배인 32px(2rem)로 
  폰트 크기를 변경한다. 
[head] 
 [style]
  .btn {
   text-decoration:none;
   font-size:2rem;
 }
 [/style]
[/head]   

5. 링크의 배경 컬러를 blue와 red로 설정한다. 

[head] 
 [style]
  .btn {
   text-decoration:none;
   font-size:2rem;
 }
  .btn.blue{
   background-color:#1f75d9;
  }
 [/style]
[/head]
