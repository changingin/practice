1. let's make two botton - with [a]tag

설명 

 
1. 먼저 Blue 라는 링크와, Red 라는 링크를 각각 만든다 

  [a href="#blue"] blue [/a]
  [a href="#red"] red [/a]
  
2. 링크를 버튼 모양으로 만들자. 
 - 먼저 각 링크에 클래스 값을 준다. 
 
  [a class="btn blue" href="#blue"] blue [/a]
  [a class="btn red"  href="#red"] red [/a]
  
3. 버튼이 전형적인 링크처럼 보이지 않기 위해 밑줄을 없애 준다. 

[head] 
 [style]
  .btn {
   text-decoration:none;
  }
 [/style]
[/head]

결과 

   blue 
   red
